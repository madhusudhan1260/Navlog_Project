# ForeFlight Navlog Converter ✈️

ForeFlight Navlog Converter is a Python application that converts ForeFlight HTML navigation logs into a professional airline-style Operational Flight Plan (OFP) PDF format.

The system extracts flight information such as route details, fuel calculations, waypoint navigation data, alternate routes, airport frequencies, ATC flight plan, and enroute winds, then generates a clean aviation-standard PDF report.

## Features

- Convert ForeFlight HTML Navlog into PDF
- Generate Airline-style OFP / NAV LOG
- Extract departure and destination details
- Route and waypoint navigation table
- Fuel planning summary
- Aircraft weight calculations
- Alternate route information
- Airport frequency details
- ATC ICAO flight plan generation
- Enroute winds summary
- Professional A4 PDF formatting

## Technologies Used

- Python
- BeautifulSoup4
- ReportLab PDF Generator
- HTML Parsing
- File Handling

## Workflow

1. Upload ForeFlight Navlog HTML file
2. Parse required flight information
3. Structure aviation data
4. Generate formatted Operational Flight Plan PDF

## Project Structure

A Python-based aviation NAVLOG converter that extracts ForeFlight flight data and generates professional airline-style Operational Flight Plan (OFP) PDFs with route, fuel, waypoint, airport, and ATC details.
